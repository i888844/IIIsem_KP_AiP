Перемещение по пунктам производится с помощью клавиш "вверх", "вниз". Для открытия
пункта требуется нажать клавишу "ENTER" или "вправо". При нажатии этих клавиш, имея
"Выход" в фокусе, программа завершит работу. Тот же эффект будет достигнут при нажа-
тии клавиши "ESC".

При открытии пунктов "Справка", "Задание №1 - Обработка матриц", "Задание №2 - Реше-
ние геометрических задач", "Задание №3 - Задачи сортировки" производится раскрытие
вложенности, содержащей дополнительные пункты для выбора, что представлено на рису-
нке 6.2 Для перемещения по пунктам вложенности так же используются клавиши "вверх",
"вниз". Для исполнения пункта требуется нажать "ENTER" или "вправо". Для выхода из
вложенности требуется нажать "влево".
